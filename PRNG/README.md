# Maze Generator

## Description
A simple psuedo random number generator using the xorshift algorithm.
Starts with a provided seed and shifts the bits of the number multiple times, xor'ing itself by itself.
Results in a seemingly random number.

## Installation
Set the seed in PRNG.c and compile using a C compiler of your choice and run the resulting binary.

## Technologies
C standard library
