# Maze Generator

## Description
A simple maze generator written in C using the recursive backtracking (depth-first search) algorithm.
Uses recursive backtracking using depth-first search to continually carve paths from a maze that is fully walls.
Uses the Fisher-Yates shuffle to randomise directions.
## Installation
Compile mazegenerator.c using a C compiler of your choice and run the resulting binary.

## Technologies
C standard library
