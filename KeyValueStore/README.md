# Key Value Store

## Description
A bash script key-value database manager.


## Usage
The following creates the key name with the value John. 
./db.sh create name John
The outputs creates the value of the key name. 
./db.sh read name
The following creates the key name with the value John. 
./db.sh update name Mike
The following creates the key name.
./db.sh delete name

## Technologies
Bash
